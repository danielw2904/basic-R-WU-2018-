Exercise 3

  1. Create a data.frame with 100 observations of one numeric and one categorial variable with 3 groups. Give each a meaningful name!

  2. Print the numeric column using the $ and the [ operators.

  3. Calculate and save the mean of the numeric column.

  4. Change the 5th value in the numeric column to the mean of the column. Use the head function to check if it worked.

  5. Create a boxplot of the numeric variable for each group

  6. Create a line plot of the numeric values for each group. Sort the values, set a title and add a legend!

Solution 3

#1. Create a data.frame with 100 observations of one numeric and one categorial variable with 3 groups.
df <- data.frame(value = rnorm(100), 
                 group = sample(c('a', 'b', 'c'), size = 100, replace = TRUE))

#2. Print the numeric column using the $ and the [ operators.
head(df$v)
## [1]  1.271409e+00  6.449964e-01  2.099275e-02 -7.466615e-05 -3.158308e-01
## [6]  1.134936e+00

##           value

## 2  6.449964e-01
## 3  2.099275e-02
## 4 -7.466615e-05
## 5 -3.158308e-01

#3. Calculate and save the mean of the numeric column.
(m <- mean(df$value))
## [1] -0.009818701
#4. Change the 5th value in the numeric column to the mean. Use the head function to check if it worked.
df$value[5] <- m
head(df$value)
## [1]  1.271409e+00  6.449964e-01  2.099275e-02 -7.466615e-05 -9.818701e-03

#5. Create a boxplot of the numeric variable for each group
boxplot(df$value~df$group)

#6. Create a line plot of the numeric values for each group. Sort the values, add a title and a legend!
df <- df[order(df$value),]
plot(df$value[df$group == 'a'], col = 'red', type = 'l', main = "Population Growth",
     xlab = 't', ylab = '') 
lines(df$value[df$group == 'b'], col = 'blue') 
lines(df$value[df$group == 'c'], col = 'green')
legend(x = 'topleft', 
       legend = c('Artful Aardvark', 'Bionic Beaver', 'Cosmic Cow'), 
       lty = 1, col=c('red', 'blue', 'green'))
